## 简介

小石榴是一款**三方**的教务系统软件，由广东财经大学 21 级学生开发。如果你已经厌倦了那难用的网页端教务系统和庞大的套壳软件，小石榴值得一试。

小石榴 UI 基于 [Compose Multiplatform](https://www.jetbrains.com/zh-cn/compose-multiplatform/) 设计，提供查课表、查成绩、选课等功能。

小石榴完全开源，个人数据全部保存于本地，更加便捷可靠。
