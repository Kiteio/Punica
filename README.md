- Android
- Windows

## 快速开始

1. 下载软件。你可以在[发行版](https://github.com/Kiteio/Punica/releases)中下载石榴。

2. 登录账号。打开软件（记得连上校园网或开启VPN），切换到设置页面，打开教务系统账号页面，点击右上角加号，登录教务系统账号。
